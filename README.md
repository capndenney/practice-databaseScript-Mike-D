## Instructions
Create a complete SQL database creation script and test it.

The script should include:
- Three tables that have at least one foreign relationship key.
- CREATE TABLE, DROP TABLE, and INSERT statements.
- Being wrapped in a transaction.

Test Case:
- INPUT: Run your script using MySQL Workbench.
- EXPECTED OUTPUT: Your script should run without errors, and you should
see the tables created in the database with the specified data.

## Setup

For this activity, you will create your own repository to store your code:
1. Create a new repository.
2. Name it: practice-databaseScript-[Your First Name]-[Your Last Initial]
3. Make sure the repository is public.
4. Clone the repository to your local machine.
5. Start working on the code.
